How to install

  1. Go to right click -> properties on the compressed file (.zip you downloaded) -> tick the unblock checkbox (If it is in the properties window that has come up) -> click ok.
  2. Extract the compressed file (.zip you downloaded).
  3. Open the rdpccgs folder and go into either dotnet4point5(recommended if in win8+) or dotnet3point5(recommended if in win vista or below).
  4. Double click the applicaition.
  5. If it asks you to install features, install them if it is your computer you are using.

How to use

  1. Select the computer from the drop down box.
  2. Select start.
  3. Click connect, type in ccgsf\ then your user name and then type in your password where it says password.
  4. Click ok.
  5. Note, you might need to repeat steps 3 and 4.
  6. Click yes to any other prompts.
  7. More informaition on the help video playlist: Youtube Playlist.
  8. Do not forget that you can report bugs on the release video or by emailing me.
  9. Note: You are only able to login if you go to Chatham & Clarendon Grammar School.
  10. Command line arguments can be gotten by running "rdpccgs.exe" "-?" .

Troubleshooting

You must be at Chatham & Clarendon Grammar School in Ramsgate, Kent, United Kingdom to use this connection system.

  • Problem : The program/windows is asking to install .net components:

    Fix : Just install the components or download microsoft .net framework yourself!

  • Problem : The error : "Invalid connection file (ALM\rdpccgs\dotnet4point5\ccgs1.rdp) specified" shows on mstsc (Remote Desktop Connection Client) when running a connection:

    Fix : Delete data.txt and all the .rdp (remote desktop connections) connections in the program's folder, then force an update. (Or just re-extract the program)

  • Problem : The message : "The publisher of this remote connection cannot be identified. Do you want to connect anyway?" shows on mstsc (Remote Desktop Connection Client) when running a connection:

    Fix : Just tick the checkbox for "Don't ask me again for connections to this computer" and hit the connect button!

  • Problem : The message : "You must change your password before logging on the first time. For assistance, contact your system administrator or technical support." or something similar (such as password expired) shows on mstsc (Remote Desktop Connection Client) when running a connection:

    Fix : Reset your password using mail.ccgrammarschool.co.uk.

  • Problem : The message : "Type a valid Remote Desktop Gateway server address. Your computer can't connect to the remote computer because the Remote Desktop Gateway server is temporarily unavailable. Try reconnecting later or contact your network administrator for assistance." shows on mstsc (Remote Desktop Connection Client) when running a connection:

    Fix : Just keep trying to reconnect and check intranet.ccgrammarschool.co.uk for school network status.

  • Problem : The message : "Your account has been disabled. Please contact a system administrator." shows on mstsc (Remote Desktop Connection Client) when running a connection:

    Fix : Go up to the administrators and ask them why!

  • Problem : A message starting with : "Remote desktop cannot find the computer" shows on mstsc (Remote Desktop Connection Client) when running a connection:

    Fix : Tell me about this (give the computer/connection name) so the connection system can be updated, use the youtube video comments or github.
